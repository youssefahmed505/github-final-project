# Simple Interest Calculator Project

## Description
This project is a simple interest calculator built using a Bash script. It calculates the simple interest based on the principal amount, the annual rate of interest, and the time period in years.

## Formula
Simple Interest = (Principal * Rate * Time) / 100

## How to Use
1. Run the `simple-interest.sh` script.
2. Enter the principal amount when prompted.
3. Enter the rate of interest per year.
4. Enter the time period in years.
5. The script will output the calculated simple interest.
